# Contributing

Helmrail is public source, but not open contribution yet.

We are not accepting:

- pull requests
- feature requests
- support issues
- roadmap requests
- connector requests

Why: Helmrail is still in its architecture-stabilization phase. Premature contribution intake would create maintenance, support, security, and governance load before the core wrapper/orchestration and data-contribution boundaries are stable.

You are welcome to inspect, self-host, and experiment locally under the project license. Please do not open PRs or issues yet; they will be closed without review.

Contribution channels may open later once the project has stable governance, connector rules, and anonymized data-contribution workflows.
